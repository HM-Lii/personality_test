# 镜像题语义同向证明

> 对 `MIRROR_PAIRS` 中 10 对镜像题，逐对回答：两题是否测量同一取向？
> 不能仅靠「都是 O 维场景」，需证明高分端/低分端在两题对应同一构念。
> 配套文件：`docs/construct-matrix.csv` 的 `primary_facet` / `cross_load_risk` 列。

## 评判格式

每对给出：两题场景、+3/-3 在两题的对应行为、同向结论、facet 标注、风险备注。

结论分三档：
- **保留**：同向成立，无需改动。
- **保留 + 微调文案**：同向成立，但某题某选项有轻度串维，任务 5 改文案即可。
- **换题**：同向不成立，需替换其中一题。

## O1 – O4：对新方法/新工具的态度

| | O1 新工具到了，旧方法还能用 | O4 现有流程稳定，有人提新做法 |
| --- | --- | --- |
| +3 | 全用新工具，边做边学 | 愿意花时间磨合，尽快跑新方法 |
| -3 | 先用老办法，以后再看 | 老流程没问题，先不改 |

**同向**：两题 +3 都=主动承担迁移成本投入新做法，-3 都=沿用稳定熟悉路径。测同一取向（Actions 尝新）。

**facet**：O1=Actions，O4=Actions。同 facet。

**结论**：保留。场景不同（个人选工具 vs 团队流程），构念一致。

## O2 – O5：学习/探索范围宽窄

| | O2 陌生城市完整一天 | O5 短时间补充陌生领域知识 |
| --- | --- | --- |
| +3 | 只定大方向，路上想去哪去哪 | 同时看几种不同说法，比较区别 |
| -3 | 去几个口碑好的地方，轻松逛完 | 只看靠谱入门资料，只学眼前要用的 |

**同向**：两题 +3 都=开放发散、拥抱多种可能，-3 都=窄而聚焦、依托已知与口碑。测同一取向（开放探索广度）。

**facet**：O2=Actions（行动探索），O5=Ideas（认知探索）。不同 facet，但都属 O 的开放取向——O2 测行动层面的探索广度，O5 测认知层面的探索广度。

**结论**：保留。facet 不同但构念同向，互补而非冲突。

## C1 – C5：无外部监督下的自我驱动结构

| | C1 两周交付由你安排 | C5 长期目标迟迟无反馈 |
| --- | --- | --- |
| +3 | 把步骤和时间都排好，留点余地 | 给自己定阶段目标，按进度推 |
| -3 | 先做最想做的，后面边做边想 | 按兴趣和状态做，不固定节奏 |

**同向**：两题 +3 都=自建结构与目标推进，-3 都=凭兴趣状态即兴。测同一取向（自我驱动结构）。

**facet**：C1=Self-Discipline（结构化自律），C5=Achievement Striving（目标推进）。不同 facet，都属 C 的自我驱动——C1 偏结构搭建，C5 偏目标持续推进。

**结论**：保留。facet 互补，构念同向。

## C2 – C4：个人事务的组织程度

| | C2 半天空闲堆着几件事 | C4 多日出行做准备 |
| --- | --- | --- |
| +3 | 每件事分好时间，到点就换 | 照着清单提前收好，出发前再核对 |
| -3 | 看当时想做什么，先挑一件开始 | 出发前看天气和心情，快速收拾 |

**同向**：两题 +3 都=提前结构化组织，-3 都=临场即兴。测同一取向（Order）。

**facet**：C2=Order，C4=Order。同 facet。

**风险**：C2 的 D 选项「先做最影响心情的」引入 R 的情绪驱动（矩阵标 yellow）。C4 较干净。

**结论**：保留 + 微调文案。任务 5 把 C2 的 D 选项「最影响心情的」改为「最在意的」之类，去掉情绪锚定，使两题纯度对齐。

## E1 – E5：社交/公开场合主动程度

| | E1 没有熟人的聚会 | E5 成果需要公开介绍 |
| --- | --- | --- |
| +3 | 主动加入聊天，帮大家互相认识 | 愿意自己上台，把成果讲给大家听 |
| -3 | 先在旁边看看，等有人自然聊过来 | 更想准备材料，让更爱上台的人去讲 |

**同向**：两题 +3 都=主动出击承担社交/展示，-3 都=回避被动等待。测同一取向（E 的主动驱动）。

**facet**：E1=Gregariousness（社交合群），E5=Assertiveness（公开表达）。不同 facet，都属 E 的主动驱动——E1 偏社交场合合群，E5 偏公开场合表达。

**结论**：保留。facet 互补，构念同向。

## E2 – E3：是否从人际互动中获取能量（最高风险对）

| | E2 弄清复杂新问题 | E3 连续忙碌后的自由周末 |
| --- | --- | --- |
| +3 | 边聊边想，慢慢理清看法 | 去热闹的地方或参加活动，给自己充电 |
| -3 | 先自己想明白，再找人确认重点 | 留大块时间独处，尽量少回消息 |

**同向论证**：两题都测「是否从人际互动中获取/补充能量」这一 E 核心构念。
- E2 的 +3=通过交谈加工思路（社交认知获能），-3=独处加工后才有底气交流；
- E3 的 +3=从热闹社交恢复（社交充电），-3=从独处恢复。
- 高分端都=社交获能，低分端都=独处获能。**同向成立**。

**facet**：E2=Gregariousness，E3=Activity。不同 facet，都属 E 的社交能量取向。

**两个风险点**：
1. E2 的「边聊边想」对 O 高分者也有吸引力（Ideas 通过对话激发），可能串 O（矩阵标 yellow，forbidden_load=O）。
2. E3 的 context「精神上需要恢复」用「恢复」一词，易与 R 的情绪恢复混淆（矩阵标 yellow，forbidden_load=R）。

**结论**：**保留 + 微调文案**（不换题）。同向成立，风险可通过文案控制：
- E2：在 context 里点明「你更倾向自己想清楚，还是借讨论想清楚」，弱化「思路激发」的 O 联想，把选项导向「加工介质是独处还是交谈」。
- E3：把「精神上需要恢复」改为「精力需要补充」，把 A 选项「给自己充电」保留，强化「能量来源」而非「情绪恢复」，与 R 划清边界。

若任务 5 微调后仍觉 E3 与 R 边界模糊，备选方案是换 E3 为一道更纯的「社交获能」题（如「长期独处后你会主动找人吗」），但当前倾向保留。

## A1 – A5：冲突中合作 vs 原则

| | A1 会议方向性分歧 | A5 与熟人协商双方在意的安排 |
| --- | --- | --- |
| +3 | 先找共同目标，想两边都能接受的办法 | 把两边最在意的都列出，再想新办法 |
| -3 | 支持我觉得更靠谱的一方，尽快定下来 | 直接说出我的方案，让对方决定接不接受 |

**同向**：两题 +3 都=寻求双赢/兼顾双方，-3 都=坚持己方立场。测同一取向（Compliance 协作顺和）。

**facet**：A1=Compliance，A5=Compliance。同 facet。

**结论**：保留。场景不同（会议分歧 vs 熟人协商），构念一致。

## A2 – A4：对他人的信任/支持 vs 直接/审慎

| | A2 朋友讲起你不看好的决定 | A4 新同伴希望负责关键部分 |
| --- | --- | --- |
| +3 | 先听对方说，没问我就不扫兴 | 把目标和支持说清楚后，放心交给他 |
| -3 | 直接说出最大问题，免得对方误会 | 关键任务还是给熟手，先少冒风险 |

**同向**：两题 +3 都=信任/支持他人（不扫兴、放权），-3 都=直接/风险控制（指出问题、依赖熟手）。测同一取向（A 的高分端=配合信任）。

**facet**：A2=Compliance（顺和不对抗），A4=Trust（信任放权）。不同 facet，都属 A 的高分端——A2 偏顺和，A4 偏信任。

**风险**：A4 的 C 选项「先少冒风险」引入 R 的风险敏感（矩阵标 yellow，forbidden_load=R）。

**结论**：保留 + 微调文案。任务 5 可把 A4 的 C 选项「先少冒风险」的措辞从风险导向改为经验导向（如「先让有经验的熟手担关键部分」），弱化 R 联想。A2 较干净。

## R1 – R4：挫折后恢复节奏

| | R1 重要安排突然被取消 | R4 投入很久的尝试未达预期 |
| --- | --- | --- |
| +3 | 缓一缓，很快开始找新办法 | 先恢复正常生活，过几天再回头总结 |
| -3 | 先想清会损失什么、还有哪些风险 | 要缓一阵，暂时不想碰类似的事 |

**同向**：两题 +3 都=快速恢复转向，-3 都=被挫折占用（盘算损失/回避）。测同一取向（Vulnerability 恢复力）。

**facet**：R1=Vulnerability，R4=Vulnerability。同 facet。

**结论**：保留。场景不同（安排取消 vs 尝试失败），构念一致。

## R2 – R3：对负面/不确定信息的情绪占用

| | R2 收到措辞直接的批评 | R3 等待无法催促的重要结果 |
| --- | --- | --- |
| +3 | 能分开语气和内容，很快开始处理 | 照常过日子，结果出来再说 |
| -3 | 会一直想那几句话，过一阵才听得进内容 | 会不断找消息，想提前猜到结果 |

**同向**：两题 +3 都=不被信息占用、维持节奏，-3 都=反刍/搜寻被占用。测同一取向（Anxiety 情绪占用，反向）。

**facet**：R2=Anxiety，R3=Anxiety。同 facet。

**结论**：保留。场景不同（批评 vs 等待），构念一致。

## 对 facet 覆盖的连带影响

- E2–E3 保留且 facet 分别为 Gregariousness / Activity，E 维 facet 覆盖维持 3 个（Gregariousness、Activity、Assertiveness）。
- O 维的 facet 覆盖已通过重新认定 O3 为 Fantasy 解决（见 `construct-matrix.csv`）：O3 的 context 明确测「未来生活可能变成什么样」，属对未来可能性的想象，归 Fantasy；其「不急着下结论」带 Ideas 成分作 secondary。O 维现为 Actions×3 + Fantasy×1 + Ideas×1 = 3 facet。
